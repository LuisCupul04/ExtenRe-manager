# 👋 Contribution guidelines

This document describes how to contribute to ExtenRe Manager.

## 🐞 Submitting a bug report

If you encounter a bug while using ExtenRe Manager, open an issue using the
[Bug report issue template](https://github.com/LuisCupul04/ExtenRe-manager/issues/new?assignees=&labels=Bug+report&projects=&template=bug_report.yml&title=bug%3A+).
