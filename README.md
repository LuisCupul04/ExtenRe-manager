# 💊 Extenre Manager

![GitHub Workflow Status (with event)](https://img.shields.io/github/actions/workflow/status/LuisCupul04/extenre-manager/release.yml)
![GPLv3 License](https://img.shields.io/badge/License-GPL%20v3-yellow.svg)

Application to use ExtenRe on Android

## ❓ About

Extenre Manager is an application that uses [Extenre Patcher](https://github.com/LuisCupul04revanced-patcher) to patch Android apps.

## 💪 Features

Some of the features Extenre Manager provides are:

- 💉 **Patch apps**: Apply any patch of your choice to Android apps
- 📱 **Portable**: Extenre Patcher that fits in your pocket
- 🤗 **Simple UI**: Quickly understand the ins and outs of Extenre Manager
- 🛠️ **Customization**: Configurable API, custom sources, language, signing keystore, theme and more
